HTML + CSS + JavaScript Self Assessment

10 tags in HTML

  • <!DOCTYPE> tells the browser about the HTML version of the page.
  • <html> is the root element of an HTML page.
  • <head> contains important information and styling of the page.
  • <body> contains everything that's going to be displayed on the page.
  • <div> is a generic container that can be used for styling and scripting.
  • <span> is an inline container used for styling a specific piece of content.
  • <h1> - <h6> are header tags used for defining headings, with <h1> being the highest level.
  • <p> is used to represent a paragraph.
  • <a> is used for hyperlinks.
  • <meta> provides metadata about the content of the page.

Header Tags for SEO

  • <h1> represents the main title of the page and is critical for SEO.
  • <h2> often represents subsections or related content to the <h1> tag.
  • <h3> - <h6> provide further subdivisions of content, helping search engines understand the hierarchy of information on the page.

different font sizes

  • 20px - Used when precise control over text size is needed.
  • 1em - Relative to the font-size of the element's parent, used for scalable designs.
  • 1.5em - 50% larger than the parent's font-size, used for highlighting text.
  • 100% - Same size as the parent's font size, used for consistency.
  • 2rem - Relative to the root element's font size, used for consistent scaling.
  • 16pt - Used for print media, not recommended for screen display.
